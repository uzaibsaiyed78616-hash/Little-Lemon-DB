# Little Lemon Database Project

## Overview
This project demonstrates a complete restaurant database system including booking management, stored procedures, and data visualization.

## Features
- MySQL database schema
- Stored procedures (booking management)
- Python client integration
- Tableau dashboard for data analysis

## Tools Used
- MySQL
- Python
- Tableau

## Files Included
- schema.sql
- procedures.sql
- app.py
- er_diagram.png
- dashboard.png
